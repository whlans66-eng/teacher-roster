## ✨ 連動效果預覽

### 當您新增一門課程時：

```
1. 填寫課程資料 ✍️
   ↓
2. 自動檢查衝堂 ⚠️ (如有衝突會警告)
   ↓
3. 新增到派課管理 ✅
   ↓
4. 自動同步到行事曆 📅 (相同ID，使用課程顏色)
   ↓
5. 更新本週課表 📋 (側邊欄立即顯示)
   ↓
6. 更新統計圖表 📊
   ↓
7. 保存到 localStorage 💾
```

### 當您刪除一門課程時：

```
1. 點擊刪除按鈕 🗑️
   ↓
2. 從派課管理移除 ❌
   ↓
3. 從行事曆自動移除 ✅ (使用相同ID匹配)
   ↓
4. 更新本週課表 📋
   ↓
5. 保存變更 💾
```

### 當您切換到行事曆時：

```
點擊「📅 行事曆」按鈕
   ↓
顯示月曆視圖，包含：
  • 所有派課（自動同步）
  • 藍色 = 正課
  • 黃色 = 補課
  • 紫色 = 實驗課
  • 綠色 = 實作課
  • 粉色 = 專題
```
