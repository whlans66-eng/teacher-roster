② 船端主系統 SHIP CORE
📊 Dashboard 儀表板 核心
  • 期初/期末結餘（USD/TWD）
  • A216 船用金彙整
  • V科目總支出彙整
  • A217 台籍/外籍分開統計
  • 即時現金餘額計算
💵 A216 船用金 CASH ADVANCE
  • 港口別預支登錄
  • USD/TWD 雙幣換算
  • 獨立匯率（A216 Rate）
  • 總表明細下載（蓋船章）
🍽️ V200 伙食費 MEAL ALLOWANCE
  • 批次計算（日期區段×名單）
  • 連動船員上下船日期取交集
  • 年節加菜金（春節3天鎖定）
  • 區段重疊防呆
🚌 V802 交通費 TRANSPORTATION
  • 個人關聯（連動船員名單）
  • 計程車免憑證，其餘強制上傳
  • 港口代碼強制 5碼 大寫
  • 簽收單 + 頁腳四格簽核
🏥 V804 醫療費 MEDICAL
  • 體檢/就醫/疫苗/其他 分類
  • 體檢品名下拉（3選1）
  • 摘要格式強制：城市+醫院
  • AI 收據辨識 + 簽名檢核
🥂 V810 交際費 ENTERTAINMENT
  • 碼頭招待 / 招待使用 / PSC
  • 菸品管控邏輯
  • AI誤差超1%警告
  • 4 KEYMAN + MASTER 簽核
📦 V401/V402 物料採購
  • 甲板/機艙急缺物料
  • AI 上傳辨識日期/金額/項目
  • SUPPORTING → 整合 A274
🎉 V819 船員活動費
  • 集體活動（不關聯個人）
  • 支援多張發票上傳
  • MASTER 簽名彙整表
🧾 V911 雜費 MISC
  • 茶葉/香燭/圖書雜誌
  • AI上傳辨識
  • MASTER 蓋船章
🏆 A217 獎金/薪資模組 BONUS
  • 總覽：定義項目 + 核准總額
  • 分配：個別船員分配金額
  • 矩陣總表（縱:船員 橫:項目）
  • 超額/未分配完 → 阻擋送出
  • 台籍/外籍幣別自動切分
🧮 A274 稅務整合 TAX
  • 各科目 SUPPORTING 按鍵→帶入
  • 台灣開立單據稅額計算
  • 定位點欄位（IMPORT C）
📤 報表產出
  • PDF：各科目簽收單（船員簽名欄）
  • Excel：定位點輸出（WDPCM116/PSF713）
  • 送回前全檢（阻擋未完成項目）
  • 匯出前乾跑驗證（筆數+加總）

④ 岸端系統 SHORE
📥 資料接收中心
  • 接收船端月底整包資料
  • 檔案版本管控
  • 各船帳狀態追蹤（進度看板）

🤖 AI 自動審核 開發中
  • OCR 讀取：日期/廠商/金額
  • 與輸入值比對誤差 >1% 警告
  • 簽名筆跡檢核（簽收單）
  • 歷史異常值警示
🔍 綜合搜尋分析
  • 全船隊各科目金額查詢
  • 交際費菸品消耗量趨勢
  • 醫療費/交通費異常偵測
  • 伙食帳廠商/港口/日期查詢
✅ 岸端審核作業
  • 承辦人逐筆審核確認
  • 主管電子簽核
  • 退回補件（附說明通知船端）
  • 列印驗收單供主管蓋章
💰 獎金核准流程
  • 公務部/海技部輸入核准總額
  • API 推送 → 船端 A217 帶入
  • 取代 Email 手動輸入流程

  • WDPCM116 IMPORT C（A274稅務）

  • 定位點驗證 → 乾跑 → 匯入

  • AI 分析摘要建議
⚡ ③ API 串接層 — INTEGRATION LAYER 規劃中
👥 FMS 船員名單 Crewlist API
  • 即時抓取上/下船日期
  • 職稱 (Rank) / 姓名
  • 員編判別台籍/外籍
  • V200/V802/V804/A217 共用
  • 降級：CSV手動匯入
💱 匯率系統 Exchange Rate API
  • 每月27日岸端財務發布
  • 全域匯率 (Global Rate)
  • A216 獨立匯率分開管理
  • Redis快取，帶時間戳版本
  • 降級：管理員手動輸入
🏅 獎金核准 API Bonus Approval
  • 公務部/海技部核准後觸發
  • 自動帶入 A217 核准總額
  • 取代 Email 手動流程
  • 降級：岸端手動輸入推送
🧠 AI OCR 服務 Claude API
  • 多模態讀取 PDF/JPG 單據
  • 擷取：日期/廠商/金額
  • 簽名筆跡檢核
  • BullMQ 非同步佇列
  • 結果人工確認後才入庫
🔒 身份驗證 Auth / SSO
