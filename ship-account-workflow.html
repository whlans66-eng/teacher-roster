🚧 關鍵防呆機制（阻擋點）
📝 輸入層防呆
  • 港口代碼強制5碼大寫
  • 計程車以外強制上傳單據
  • V804體檢摘要強制格式（城市+醫院）
  • 伙食費區段不可重疊
  • 天數不可為負，最少1天
  • A217分配不可超過核准總額
🚫 送出層阻擋
  • A217任一項目Remaining > 0 → 阻擋
  • 必填欄位缺漏 → 阻擋下載簽收單
  • 體檢摘要格式錯誤 → 阻擋下載
  • 非計程車未上傳憑證 → 阻擋下載
  • Excel乾跑筆數/加總不符 → 阻擋匯出
  • 名單姓名/職稱為空 → 阻擋V200簽收單
⚠️ AI警示層
  • AI讀取金額 vs 輸入金額誤差>1% → 警告
  • 簽收單有紀錄但無簽名筆跡 → 紅色警告
  • 單據日期與錄入日期不符 → 警示
  • 交際費菸品消耗量異常 → 岸端標示
  • 歷史金額異常偏高 → 岸端提示
  • 農曆春節鎖定3天，端午/中秋鎖定1天
📡 離線與弱網策略（船端核心問題）
✅ 操作不依賴網路
  • 所有船端輸入先存入 IndexedDB
  • 離線狀態顯示「本地暫存」標記
  • 可完整填完整份船帳再一次上傳
  • 操作紀錄 idempotent，可重播
🔄 網路恢復後同步
  • Service Worker 背景自動同步
  • 版本號衝突時提示使用者確認
  • 同步完成後標記「已同步」
  • 每次同步保留完整操作紀錄
⚡ API 降級備援
  • FMS無回應 → 手動匯入CSV船員名單
  • 匯率API失效 → 管理員手動輸入鎖定
  • 獎金API失效 → 岸端手動輸入後推送
  • AI服務中斷 → 人工手動填寫繼續作業
